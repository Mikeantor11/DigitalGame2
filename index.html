Reaction Time!

  • High Concept: Press a button as soon as the screen changes color!
  • Player Experience Goals: Work on developing a faster reaction time.
  • Genre: Reaction
  • Game play: The player needs to press enter to begin the random delay for how long until the screen changes. Once the screen changes green press the space bar as soon as possible!
  • Prototype goal: Test how a reaction based game will appeal to players.
  • Features: The user will get the opportunity to improve their reaction time!
  • Setting/Premise: There is no setting for this game!
  • Story: There is no story for this game!
  • Target Audience: Casual players who want to set a high score!
  • Play time: Each round takes about 5 seconds, you can play as many rounds as you would like!
  • Strategies: Press the spacebar as fast as possible.
  • Prototype Assets: Everything is this game was made this Phaser.
  • Prototype Closest Other Game: Reaction based games.
